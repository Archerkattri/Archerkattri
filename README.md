## ▌ In the lab

🫳 **GaussianFeels** : real-time visuo-tactile 3D-Gaussian SLAM for in-hand manipulation. One object-centric Gaussian map serves tracking, reconstruction, rendering, and manipulation-facing geometry at once. **<!-- AUTOGEN:gaussianfeels.add_s_sim_mm -->0.83 mm<!-- /AUTOGEN:gaussianfeels.add_s_sim_mm -->** ADD-S in simulation, **<!-- AUTOGEN:gaussianfeels.add_s_real_mm -->3.37 mm<!-- /AUTOGEN:gaussianfeels.add_s_real_mm -->** on real hardware, about **<!-- AUTOGEN:gaussianfeels.speedup_vs_neural_field -->7.6×<!-- /AUTOGEN:gaussianfeels.speedup_vs_neural_field -->** faster than the neural-field baseline, **<!-- AUTOGEN:gaussianfeels.sim2real_retention -->94%<!-- /AUTOGEN:gaussianfeels.sim2real_retention -->** sim-to-real reconstruction retention.
